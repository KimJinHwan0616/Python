># 자료형 *(Data Type)*
>자료의 `형태`
>
>### 확인, 변환
자료 *(data)*: `프로그램`이 `처리`하는 모든 것(≒리터럴)

인덱스 *(index)*: `메모리`에 저장된 `자료`의 `순서`
```angular2html
예) 문자열, 숫자, 불, 리스트, 튜플, 사전, 집합
```
---

## 확인
`type(자료형)`


## 변환

+ ### 문자열 → 숫자
  >(정수)문자열 → 정수: `int( "(정수)문자열" )`
  >```
  >int( "52" )
  >```
  >
  >(실수)문자열 → 실수: `float( "(실수)문자열" )`
  >```
  >float( "52.283" )
  >```
  
+ ### 숫자 → 문자열
  `str(숫자)`
