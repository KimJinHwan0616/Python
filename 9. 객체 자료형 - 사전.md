## 사전 *(dictionary)*
`{'키1':값1, '키2':값2, ... '키n':값n}`
```
a = {'이름': '홍길동', '나이': 32} 
```

+ ### 연산자
  >선택: `사전['키']`
  >```
  >a['이름']    # 홍길동
  >a['나이']    # 32
  >```
  >
  >추가: `사전['키'] = 값`
  >```
  >a['성별'] = '남자'    # a = {'이름': '홍길동', '나이': 32, '성별': '남자'} 
  >```
  >
  >제거: `del 사전['키']`
  >```
  >del a['이름']    # a = {'나이': 32}
  >```
  >
  >검색: `'키' in 사전`
  >```
  >'생년월일' in a    # False
  >'이름' in a    # True
  >```
