## 입력
`input( (프롬포트)문자열 )`
```angular2html
input("입력하세요")    # 입력하세요
```

## 출력 *(문자열)*
`printf(자료형1, 자료형2, ..., 자료형n)`

+ ### 여러 줄 문자열
  `print('''\문자\''')`
  ```angular2html
  print('''\
  안녕하세요 
  안녕하세요
  \''')
  ```
